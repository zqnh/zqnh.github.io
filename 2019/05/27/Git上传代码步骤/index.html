Git上传代码步骤

发表于 2019-05-27 | 分类于 GitHub | | 
代码上传到GitHub步骤

一、首次上传
1.安装Git(安装成功后右击文件目录会显示两个Git操作)
2.配置身份
git config –global user.name “用户的名字”
git config –golbal user.email “邮箱”
3.在项目目录下右击Git Bash Here 如图：

4.输入git init 命令 命令的意思是:使之成为本地git仓库
5.git add . 把本地项目里的所有文件添加到刚刚生成的本地git仓库
6.git commit -m “这里是提交时候添加的注释”
7.git remote add origin git@github.com(你刚创建项目后的SSH地址)
8.git pull origin maser //把本地仓库更新和远程仓库一样
9.git push -u origin master提交文件
10.等几秒会提示叫你输入密码：Enter passphrase for key ‘/Users/renpeng/.ssh/id_rsa’: 就是安装Git时的密码
最后:我们在github上的项目文件就可以看见我们上传的文件了.
11.如果以前连接过，从第三步开始
12.如果首次连接会提示连接SSH进行
打开GitHub——>点击头像Setting——>SSH and GPG keys——>New SSH key

title随便填写，key是在C盘用户目录里找到id_rsa.pub文件，用记事本打开，然后复制内容。保持就ok了
